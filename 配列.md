# 配列

## 拡張For文
配列やコレクションと呼ばれる複数の要素を持っているものから全ての要素に含まれる値を順に取り出して処理するために使われる反復構造。
```java
for("変数": 配列){

}
```
for文と拡張for文の違いを見ていく。

```java
int[] points = { 64, 76, 58, 72, 48 };

//普通のfor文
for (int i = 0; i < points.length; i++) {
			System.out.println(points[i]);
		}

//拡張for文
for (int i : points) {
			System.out.println(i);
		}
```
